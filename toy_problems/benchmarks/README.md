# merlion
MERLION: a multi-objective meta-reinforcement learning framework for learning diverse policies across preference weights and tasks.

---

This is directory of benchmark merlion for toy problems

---

## Environment Setup

This project uses **Python 3.11.13** and dependencies listed in `requirements.txt`.

### 1. Create Conda Environment

```bash
conda create -n merlion_benchmark python=3.11.13

```

run run_morl.py and select the algorithms you want to train
don't forget to login to wandb to get the results

### 2. Activate Environment
```bash
conda activate merlion_benchmark
```

### 3. Install Dependencies
```bash
pip install -r requirements.txt
```
